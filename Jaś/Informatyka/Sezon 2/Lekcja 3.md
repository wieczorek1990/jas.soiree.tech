# Programy tekstowe

Listowanie argumentów:

```python
#/usr/bin/env python
import sys


# Strip the program name, the first argv item.
arguments = sys.argv[1:]
# Print arguments.
print(arguments)
```

Pozdrawiacz:

```python
#/usr/bin/env python
import sys


name = sys.argv[1]
print(f"Hello {name}.")
```

Drukowanie tekstu:

```python
print("Text here.")
```

Pozyskiwanie danych tekstowych:

```python
answer = input("Please give me your name: ")
```

Pętla nieskończona z wyjściem:

```python
while True:
    break
```

Pętla nieskończona w wyjściami warunkowymi:

```python
while True:
    if early_condition:
        break
    ...
    if middle_condition:
        break
    ...
    if late_condition:
        break
```

Pętla nieskończona z pasowaniem:

```python
while True:
    pass
```
